# 🎨 ASCII Art Studio

A lightweight and user-friendly tool to convert your images into stunning ASCII art. Built with Python and CustomTkinter.

## ✨ Key Features
* **Modern UI**: Clean dark-themed interface.
* **Messenger-Ready**: Automatically formats ASCII art with ```code blocks``` for perfect display in Discord and Telegram.
* **High-Quality Export**: Save your art as a sharp PNG image on a dark background.
* **Real-time Tuning**: Adjust detail levels with a slider and see changes instantly.

## 🌍 For Non-Russian Speakers
Since the interface is currently in Russian, here is a quick translation guide:
* **Выбрать картинку** — Open Image
* **Скопировать для ТГ/Discord** — Copy for Telegram/Discord (with formatting)
* **Сохранить .PNG** — Save as PNG
* **Сохранить .TXT** — Save as Text file
